# LANClipSync — 局域网剪贴板同步

## 这是什么

LANClipSync 是一个 Windows 局域网剪贴板同步工具。装在两台（或多台）Windows 电脑上，它们连接同一个局域网后，其中一台电脑复制的内容会自动同步到另一台的剪贴板。

**举例**：你在台式机上复制了一段文字或截了个图，转身在笔记本上直接 Ctrl+V 就能粘贴出来。不需要 U 盘、不需要微信传文件、不需要登录任何账号。

## 工作原理

```
电脑 A 复制 → A 的剪贴板 → LANClipSync 检测到变化
                              ↓
                        压缩 → 加密 → UDP 广播发现对端 → TCP 发送
                              ↓
电脑 B 收到 → 解密 → 解压 → B 的剪贴板 → B 可直接 Ctrl+V
```

- **纯局域网**：数据不经过互联网，不经过任何服务器
- **端到端加密**：支持设置共享密钥，传输全程加密
- **自动发现**：同一局域网下的设备自动发现，无需手动配置 IP

## 技术特点

- **零外部依赖**：只用了 Python 标准库，不需要安装任何第三方包
- **Windows 原生**：使用 Win32 API 操作剪贴板，支持剪贴板监听（无需轮询）
- **单文件 exe**：打包成单个 exe 文件，双击即用，无需安装
- **绿色免安装**：不写注册表，不添加开机启动，不创建系统服务

## 适用场景

- 台式机 ↔ 笔记本 协同工作
- 办公室局域网内快速分享文本、图片、文件
- 开发环境下复制代码片段到另一台机器
- 不想用第三方云服务传输敏感信息
