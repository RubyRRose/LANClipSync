# LANClipSync（代码完全由AI编写，我只进行了提需求和把控需求，代码一行没看，放这里纯属当仓库保存）

> 局域网剪贴板同步工具 — 复制粘贴，跨设备同步

LANClipSync 让同一局域网下的 Windows 电脑共享剪贴板。在台式机上复制的内容，转身在笔记本上就能粘贴。

## 快速开始

1. 下载 `LANClipSync.exe`
2. 两台电脑都双击运行
3. 点击「▶ 点击启用同步」
4. 在设备列表中选中对方，点击「连接选中设备」
5. 开始使用：一台复制，另一台 Ctrl+V

> 可选：在两端输入相同的共享密钥开启加密传输。

## 功能一览

| 类型 | 支持 | 说明 |
|------|------|------|
| 文本 | ✅ | 代码、URL、JSON 等，50MB 上限 |
| 图片 | ✅ | 截图、复制图片（DIB 格式） |
| 文件 | ✅ | 文件、文件夹，保持目录结构 |

- **同步/手动** 双模式
- **隐私模式** — 发送前弹窗确认
- **类型过滤** — 独立开关文本/图片/文件
- **剪贴板历史** — 最近记录可恢复
- **传输加密** — 共享密钥 + XOR 流加密
- **自动发现** — 同一局域网自动找对端
- **传输进度** — 实时速度 + 进度条
- **零依赖** — 纯 Python 标准库，单 exe 文件

## 系统要求

- Windows 7/10/11（64 位）
- 两台电脑需在同一局域网
